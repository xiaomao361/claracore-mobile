# ClaraCore Mobile Visual Assets

Initial visual assets for ClaraCore Mobile.

## Brand Source Of Truth

ClaraCore Mobile should reuse the ClaraCore Desktop brand mark.

- `brand/claracore-mark.svg`: full Desktop mark with gradient and shadow.
- `brand/claracore-mark-flat.svg`: flat mark for smaller digital use.
- `brand/claracore-symbol-mono.svg`: single-color symbol.
- `brand/claracore-wordmark.svg`: wordmark.
- `brand/claracore-desktop-icon-1024.png`: generated Desktop app icon.
- `brand/claracore-desktop-icon-flat-1024.png`: generated flat Desktop app icon.
- `brand/claracore-wordmark-1800x520.png`: generated wordmark export.

The mobile app icon should be generated from the Desktop mark or a mobile-safe crop of it. Do not introduce a separate mobile-only logo unless the Desktop brand system is intentionally revised.
